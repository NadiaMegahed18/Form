# Interactive Form Project 📝

A clean and user-friendly web form designed with focus on layout structure and styling.

## 🚀 Features
* **User Interface:** Simple and intuitive design for better user experience.
* **Responsive Layout:** Adjusted to look good on different screen sizes.
* **Form Elements:** Includes various input types (text, email, buttons) with proper styling.

## 🛠️ Technologies Used
* **HTML5:** For building the form structure and semantics.
* **CSS3:** For custom styling, colors, and layout alignment.
